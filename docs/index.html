Built for real Modbus diagnostics

Dedicated tools for engineers, developers, maintenance technicians and industrial automation professionals.

Traffic Data Analyzer

View Modbus communication in real time and quickly identify communication problems.

  • Valid communication frames
  • Communication error filtering
  • CSV data export
  • Real-time diagnostics

Modbus Master & TCP Client

Communicate with industrial devices using the most common Modbus protocols.

  • Modbus RTU
  • Modbus TCP/IP
  • Modbus RTU over TCP/IP
  • Multiple numeric formats
  • Universal register tables

Integrated Modbus RTU Sniffer

Monitor communication between a Modbus Master and Slave without interfering with the network.

  • Real-time frame monitoring
  • Communication diagnostics
  • CRC validation
  • Error identification

Modbus Slave Simulator

Simulate Modbus devices during software, PLC and embedded firmware development.

  • Register simulation
  • Automatic value generation
  • Exception response simulation
  • Firmware development testing
