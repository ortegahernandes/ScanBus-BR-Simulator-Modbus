Engineering tools

Built for real-world Modbus diagnostics.

Go beyond basic register reading with dedicated tools for discovering, analyzing and troubleshooting industrial devices.

01 — ANALYZE

Traffic Data Analyzer

View Modbus communication in real time and isolate intermittent communication problems without digging through raw logs manually.

  • Valid frame and error filtering
  • Real-time communication diagnostics
  • Search and data highlighting
  • CSV data export
02 — DISCOVER

Device Scanner

Discover Modbus devices and explore supported functions and accessible register areas automatically.

  • Device ID scanning
  • Function Code discovery
  • Register Address scanning
  • Exception response detection
03 — IDENTIFY

Serial RTU Scanner

Connect to an unknown Modbus RTU device and automatically search for the serial parameters required for communication.

  • Baud Rate scanning
  • Parity and Stop Bits discovery
  • Slave ID scanning
  • CRC response validation
04 — FIND

TCP/IP Scanner

Search the local network for Modbus TCP devices when the target IP address or communication port is unknown.

  • IP address scanning
  • TCP port scanning
  • Device ID detection
  • Modbus response validation
05 — VISUALIZE

Time Graph

Plot Modbus register values over time to visualize trends, oscillations and dynamic system behavior.

  • Up to 8 independent traces
  • Automatic or manual scaling
  • Multiplication factor and offset
  • Configurable time scale
06 — REVERSE ENGINEER

Wide Vision

Monitor large memory areas and quickly identify which registers change while interacting with a device.

  • Large register area monitoring
  • Automatic change detection
  • Visual register highlighting
  • Ideal for reverse engineering
07 — MONITOR

Integrated Modbus RTU Sniffer

Passively monitor communication between an existing Modbus Master and one or more Slave devices.

  • Real-time frame monitoring
  • Master and Slave identification
  • CRC validation
  • Communication diagnostics
08 — SIMULATE

Internal Slave Simulator

Emulate Modbus Slave devices directly inside ScanBus-BR for PLC, software and embedded firmware development.

  • Editable Slave memory
  • Standard Modbus functions
  • Exception response simulation
  • Communication error simulation
