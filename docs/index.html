```

Built for real Modbus diagnostics

Dedicated tools for automation professionals, developers, engineers and maintenance technicians.

Traffic Data Analyzer

View Modbus communication in real time and quickly identify intermittent communication problems.

  • Valid frame filtering
  • Communication error filtering
  • CSV export
  • Search and highlighting

Device & Network Scanner

Automatically discover Modbus devices, supported functions and accessible registers.

  • Automatic Device ID scanning
  • Function Code scanning
  • Register Address scanning
  • Exception detection

Serial RTU Scanner

Automatically search for the serial parameters required to communicate with an unknown Modbus RTU device.

  • Baud Rate scanning
  • Parity and Stop Bits
  • Slave ID discovery
  • CRC frame validation

TCP/IP Scanner

Search the local network for Modbus TCP devices when the target IP address is unknown.

  • IP address scanning
  • TCP port scanning
  • Device ID detection
  • Modbus response validation

Time Graph

Plot Modbus register values over time to analyze trends, oscillations and dynamic system behavior.

  • Up to 8 independent traces
  • Automatic or manual scale
  • Multiplication factor and offset
  • Custom time scale

Wide Vision

Quickly identify changing registers across large memory areas — ideal for reverse engineering.

  • Real-time activity monitoring
  • Automatic change detection
  • Visual register highlighting

Modbus RTU Sniffer

Passively monitor communication between an existing Modbus Master and one or more Slave devices.

  • Real-time frame monitoring
  • Master and Slave identification
  • CRC validation
  • RS485 A/B detection

Internal Slave Simulator

Emulate complete Modbus Slave devices directly inside ScanBus-BR without requiring external hardware.

  • All standard Modbus functions
  • Editable Slave memory
  • Exception response simulation
  • Communication error simulation
```
